# 眠気チェックアプリ

Webカメラを使って30秒間の測定で眠気レベルを判定するアプリケーションです。

## 機能

- **30秒測定**: カメラを見つめるだけで眠気レベルを判定
- **PERCLOS指標**: 米国運輸省などで採用されている標準的な疲労検出指標を使用
- **プライバシー重視**:
  - カメラ映像はローカル処理のみ
  - デフォルトでカメラ表示オフ
  - データは一切外部送信されません

## 測定について

### PERCLOS (Percentage of Eye Closure)

一定期間内で目が閉じている時間の割合を計測します。

- **PERCLOS < 15%**: 正常
- **15% ≤ PERCLOS < 20%**: やや疲労
- **PERCLOS ≥ 20%**: 疲労

### スコア

0-100点のスコアで表示されます。

- **80点以上**: 正常（緑）
- **60-79点**: 注意（青）
- **40-59点**: やや疲労（オレンジ）
- **39点以下**: 疲労（赤）
